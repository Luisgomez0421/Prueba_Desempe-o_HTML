# 🎸 Shattered Riffs – Rock Band Landing Page



## 📌 Project Overview

This project is a **responsive landing page** developed to represent the rock band *Shattered Riffs*.

The website was built to deliver a **visually engaging, modern, and well-structured experience**, showcasing the band’s identity, music style, and upcoming tour events through a strong rock-inspired design.


## 🎯 Project Features

This project includes:

* A complete landing page built with **semantic HTML5**
* A custom **CSS3 design** with a strong rock aesthetic
* A fully **responsive layout** for:



  * Desktop (>1024px)
  * Mobile devices

* Use of **Flexbox and Grid** for layout structure
* A clean and organized visual presentation

## 🧩 HTML Structure

The project is built using **semantic HTML elements** to ensure clarity and proper structure:



* `<header>` – Navigation and title  
* `<nav>` – Menu links (Events, About, Media, Contact)  
* `<main>` – Main content  
* `<section>` – Content sections  
* `<article>` – Band description  
* `<footer>` – Contact and social links  



### Included Sections:



* **Hero** – Main visual section with title and call-to-action  
* **Events** – Tour dates displayed in a table  
* **About** – Band description, influences, and values  
* **Media** – Image gallery  
* **Contact/Footer** – Social links  


## 🎨 CSS Design

The styling was fully customized to create a **rock-inspired visual identity**:



* Dark gradient background  
* Red accent color for highlights  
* Bold typography and clear hierarchy  
* Smooth hover effects and transitions  



### Layout Techniques:



* **Flexbox** – Used for header, about section, and footer  
* **Grid** – Used for the media gallery  



The CSS is structured and organized for readability and scalability.


## 📱 Responsive Design

The website adapts to different screen sizes:



* Desktop layout with horizontal structure  
* Mobile layout with stacked elements  



### Key Adjustments:



* Navigation becomes vertical on smaller screens  
* Images resize automatically  
* Grid layout adapts from 3 columns → 2 → 1
